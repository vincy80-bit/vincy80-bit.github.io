Sync & Backup Center

Google Tasks Setup Instructions:
  1. Go to the Google Cloud Console and enable the Google Tasks API.
  2. Create an OAuth Client ID (Web Application).
  3. Add http://localhost:8000 (or your hosted PWA URL) to **Authorized JavaScript origins**.
  4. Copy your Client ID, paste it above, click **Connect Google**, and grant Tasks authorization!
Wi-Fi Sync Instructions:
  1. Run node sync-server.js on your computer.
  2. Find your computer's local IP address (e.g. 192.168.1.50).
  3. Enter http://<IP>:3000 in the Server IP input above.
  4. Generate a Sync Key, then click Sync Now! Use the same Key on your phone to instantly sync both devices over local Wi-Fi.
